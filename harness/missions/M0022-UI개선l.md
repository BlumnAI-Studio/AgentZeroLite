# 요청 (Brief)

AgentZero의 UI 를 개선하고자함

먼저 펜슬 디자인을 최신으로 업데이트
- Docs/design/design.pen

그리고 개선하는 내용은 펜슬을 먼저 업데이트한후 그 기반으로 개선에 착수

개선내용
- AgentZero Lite 전체화면 하기하면 윈도우TASK영역까지 가림..미디어용전체 화면이 아닌 어플리케이션 전체화면채택
- AgentCLI 대메뉴에 AgentBot 으로 이름변경후 Bot아이콘 채택
  - 그리고 WORKSPACE 메뉴공간이 있는데 완전접기 추가 : ConsoleTab을 더 넓게 보기위한 기능임
  - 적용시 마지막 레이아웃 저장기능에 적용 다음 실행시에도 동일하게 펼쳐질수 있게지원
- 여기서 채택한 Bot아이콘은 .. 실행 아이콘에도 적용할것
 - 현재 실행아이콘은 어둡고 인식이불가... 조금 밝고 배경이 투명한 아이콘 채택해 가독성을 높일것
- Scrap 대메뉴에 실행되는 기능을 동일하게 창이 분리될수있도록 DEATACH 도입 (DockBack 기능도 유사하게 적용)
- 대메뉴 4번째 기능신설 : 하네스뷰
 - 웹뷰로 https://psmon.github.io/AgentZeroLite/Home/harness-view/#dashboard 가 우측 컨텐츠 전체 사용해 표시
 - 기본 브라우저 웹뷰 이용
- WebDebPart
- Voice Note가 있는데 우측하단의 영역이 메인인데 너무작은 공간을 차지함..우측영역을 다 사용 , No note selected가 있는 페이지는 쓸모없음 제거
